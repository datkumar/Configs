## Installation

As per [their docs](https://www.rust-lang.org/tools/install), to download `rustup` and install Rust on a UNIX system, run the command:

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

## Update your installation

```sh
rustup update
```
